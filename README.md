## 📝주요 특징
**1. 통합**
- 로그인 & 회원 가입 & ID/PW 찾기 
- 놀이공원 당일 쾌적도 제공
- 놀이공원 정보 조회 (어트랙션 / 퍼레이드 / 편의시설 / 티켓요금)
<br/>

**2. 회원**
- My page
- 티켓 예매
- 어트랙션 예약
- 어트랙션 추천 기능 / 고객의 소리
<br/>

**3. 관리자**
- 회원 & 직원 관리
-  놀이공원 정보 관리
	- 어트랙션 ON/OFF 관리
	- 어트랙션 대기열 관리
	- 퍼레이드 정보 관리  
- 티켓 예매 현황
- 놀이공원 통계 기능
	- 당월/ 전체 이용자 현황
	- 설문조사 관리 (이달의 어트랙션 순위 / 고객의 소리)

### 👑 Detail
### 1. (통합) 용용랜드 정보
![AllInfo](https://user-images.githubusercontent.com/42857790/145171390-b628e83c-35f8-482d-83df-57575601047a.gif)

#### 1-1. 어트랙션 정보
- 용용랜드의 어트랙션 정보를 보여줍니다.
- 회원 추천수를 기반으로 1-5위 [이달의 어트랙션 순위]를 상단에 표시합니다.
- 어트랙션 정보는 전체보기와 타입별로 분류를 나눠 볼 수 있으며, 한 페이지 10개의 목록을 보여줍니다.

#### 1-2. 퍼레이드 정보
- 현재 진행 중인 퍼레이드 정보를 보여줍니다.

#### 1-3. 편의시설 정보
- 음식점/ 물품보관함/ 의료실의 종류와 위치, 운영시간 정보를 보여줍니다. 

#### 1-4. 티켓요금 정보
- 어른/ 청소년/ 어린이 티켓요금과 할인카드 종류를 보여줍니다.

### 3. (관리자) 놀이공원 정보 관리
![attractionONOFF](https://user-images.githubusercontent.com/42857790/145170374-a8c6708a-d675-4f8d-a31c-feb3c1c8e9b5.gif)
#### 3-1. 어트랙션 ON/OFF 관리
- 어트랜션의 ON/OFF 관리를 할 수 있습니다.
- 원하는 어트랙션의 번호 선택 후, Y/N으로 운행 여부 관리를 진행합니다.
- 이때 `ON` 입력시 전체 어트랙션 ON 처리를, `OFF` 입력시 전체 어트랙션 OFF 처리를 수행합니다.
<br/>
<br/>
<br/>


![attractionWait](https://user-images.githubusercontent.com/42857790/145170401-b2a4b657-bedb-41da-90fc-c3e3c258a01e.gif)
#### 3-2. 어트랙션 대기열 관리
- 어트랙션의 대기열을 관리 할 수 있습니다.
- 대기열 변경을 원하는 어트랙션 번호 선택 후, 변경을 원하는 대기 시간을 입력합니다.
- 대기 시간은 최대 120분까지 입력 가능합니다.
<br/>
<br/>
<br/>


![paradeModify](https://user-images.githubusercontent.com/42857790/145170419-528f5280-ed64-4e59-801a-c6dad9df074b.gif)
#### 3-3. 퍼레이드 정보 관리  
- 퍼레이드 정보를 수정할 수 있습니다.
- 퍼레이드 이름 / 날짜 / 내용 수정이 가능하며, 수정한 내용은  `놀이공원 정보 조회 > 퍼레이드 정보`에 반영됩니다.
